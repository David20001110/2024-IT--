## 一、什麼是DRF
DRF(Django REST framework) 是一個基於 Django 的 Web 應用程式框架，它提供了一組工具和函數，用於快速構建 Web API。DRF 是一個功能強大且靈活的框架，它可以幫助開發者快速構建 Web API，並提供了許多功能，如序列化、驗證、分頁、過濾、查詢、視圖等等。
